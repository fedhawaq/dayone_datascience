## Tasks

### 1. Reverse a String

This program takes a string from the user and prints it in reverse.

Example:

```
Input: DATA EXPLORE
Output: EROLPXE ATAD
```

---

### 2. Vowel Counter

This program counts how many vowels (a, e, i, o, u) are in a string.(BOTH IN  LOWER CASE AND UPPER CASE)

Example:

```
Input: Education
Output: 5
```

---

### 3. CLI Calculator

This is a simple calculator that runs in the terminal.
You choose an option and enter numbers to get the result.

It can do:

* Add
* Subtract
* Multiply
* Divide

Example:

```
1.Add  2.Subtract  3.Multiply  4.Divide  5.Exit
Choose: 1
Enter first number: 4
Enter second number: 6
Result: 10
```

```

3. Open the folder:

```

```

4. Run any file:

```
